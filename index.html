Tim berners-Lee

Who is he ?

Berners-Lee is a British computer scientist best known for inventing the World Wide Web. In 1989, while working at CERN, he proposed a system to manage and share information via hypertext, which led to the development of the first web browser and web server. His invention revolutionized communication, information sharing, and access to knowledge, fundamentally changing the way people interact online. Berners-Lee is also a strong advocate for internet standards and an open web, contributing significantly to the development of web protocols and standards through the World Wide Web Consortium (W3C).

Timeline about him :-

  • 1955: Tim Berners-Lee is born on June 8 in London, England.
  • 1980: Proposes the "Enquire" project at CERN.
  • 1989: Writes the proposal for the World Wide Web.
  • 1990: Develops the first web browser and formalizes the Web project.
  • 1991: Launches the first website at CERN.
  • 1993: Mosaic browser is released, spreading web usage.
  • 1994: Founds the World Wide Web Consortium (W3C).
  • 1999: Wins the first Millennium Technology Prize.
  • 2004: Knighted by Queen Elizabeth II.
  • 2009: Introduces the concept of Linked Data.
  • 2012: Wins the Prince of Asturias Award.
  • 2018: Releases the "Contract for the Web".
  • 2024: Continues advocating for ethical tech use.
